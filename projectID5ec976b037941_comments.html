This is a short grammar of the Tibetan language, published in 1883 in a series of "Simplified" grammars.

The difficulty here is that it uses Tibetan (བོད་སྐད་) script, which is fairly complicated, and to make things even more complex, also mixes in Devanagari (देवनागरी) and Arabic (عَرَبِيّ) script, so we can have plenty of fun. Although the site recently converted to Unicode, we cannot yet enter those scripts directly (and most people will lack keyboards for those scripts anyway). Therefore, we will use encoding schemes, which are explained in the instructions linked to by the native names of those scripts above. If you work on this project, it is expected you will transcribe the non-Latin scripts according to those instructions. If you are not willing or not able to do so, please select one of the many other projects here. If you need help to do so, you can ask your questions in the Forum, and I will do my best to clarify things.

Please also be aware that this grammar uses German style single open and close quotes around many words: please use normal single quotation marks 'like this' to represent these.

The following special symbols can be typed as given: ˱ as [_v]; ˳ as [_o], γ as [_g], and χ as [_x] (the last two are Greek gamma and chi, and appear fairly often, so you can save a little in typing).

The below table is for quick reference only; please take care to read the full transcription instructions if you are not already familiar with Tibetan script and Wylie encoding.

If you are unsure, you can always try things in the Online Tibetan conversion tool.

Tibetan is to be typed between [BO: and ]; when you encounter a new-line in Tibetan, please end the line with backslash (\), if needed preceded with a space for the tsheg. (Otherwise, the space representing the tsheg at the end of a line will be lost.)
